# Code Book

Information about the original data fields are available in `features_info.txt` The file `tidy.txt` contains the following fields: 

## Identifiers

 - `subject` - `integer` - The ID of the test subject
   - 1 - 30
 - `activity` - `string` - The type of activity that they were performing when the measurements were taken
   - Walking
   - Walking Upstairs
   - Walking Downstairs
   - Sitting
   - Standing
   - Laying
